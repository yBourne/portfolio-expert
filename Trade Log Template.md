# Trade Log Template

> 交易 / 操作记录模板。用于让 Behavior Coach Agent 分析操作质量、纪律执行和情绪影响。

## 使用方式

每次交易后复制本模板，创建一条操作记录，例如：

`Trade Logs/2026-06-01 AAPL Buy.md`
